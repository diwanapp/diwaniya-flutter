\# سياسة الاستخدام والمحتوى لتطبيق ديوانية

\## 1. مقدمة



تهدف هذه السياسة إلى توضيح قواعد الاستخدام المقبول داخل تطبيق \*\*ديوانية\*\*، وحماية تجربة الأعضاء، والحد من إساءة استخدام الدردشة، الصور، الملفات، الصوتيات، المقاضي، المصاريف المشتركة، التصويتات، السوق، العروض، المزادات، وأي خصائص أخرى يوفرها التطبيق.



تم إعداد هذه السياسة لتكون مرجعًا واضحًا للمستخدمين والمؤسسين والمدراء، ولتساعد في الحفاظ على بيئة آمنة ومحترمة ومنظمة داخل كل ديوانية. ويُعد استخدام التطبيق موافقة على الالتزام بهذه السياسة، إضافة إلى شروط الاستخدام وسياسة الخصوصية وأي ضوابط تظهر داخل التطبيق.



\---



\## 2. نطاق هذه السياسة



تنطبق هذه السياسة على جميع أشكال الاستخدام والمحتوى داخل التطبيق، بما في ذلك:



\* الرسائل والمحادثات والردود والإشارات إلى الأعضاء.

\* الصور والملفات والصوتيات والمرفقات.

\* الألبوم والمقاضي والمصاريف المشتركة والتصويتات.

\* أسماء الديوانيات والأعضاء والأوصاف والرموز أو الصور المستخدمة.

\* السوق، العروض، الإعلانات، القوائم، المزادات، والخدمات عند توفرها.

\* طلبات الانضمام والإشعارات وآخر النشاطات.

\* أي محتوى أو تفاعل أو بيانات يضيفها المستخدم داخل التطبيق.



\---



\## 3. المبادئ العامة للاستخدام



يلتزم المستخدم عند استخدام ديوانية بما يلي:



1\. احترام أعضاء الديوانية وباقي المستخدمين.

2\. استخدام التطبيق للأغراض الاجتماعية والتنظيمية المشروعة.

3\. عدم نشر محتوى يخالف الأنظمة أو الآداب العامة أو حقوق الآخرين.

4\. عدم استخدام التطبيق للإساءة أو الاحتيال أو المضايقة أو انتهاك الخصوصية.

5\. تحري الدقة عند إضافة المصاريف أو المقاضي أو التصويتات أو أي بيانات مشتركة.

6\. احترام صلاحيات المؤسس والمدراء، واستخدامها بمسؤولية عند منحها للمستخدم.

7\. عدم محاولة الوصول إلى ديوانيات أو بيانات لا يملك المستخدم صلاحية الوصول إليها.



\---



\## 4. المحتوى الممنوع



يُمنع استخدام التطبيق لنشر أو إرسال أو رفع أو مشاركة أي محتوى يتضمن، على سبيل المثال لا الحصر:



\* محتوى مخالف للأنظمة أو التعليمات المعمول بها في المملكة العربية السعودية.

\* محتوى مسيء أو مهين أو تشهيري أو تهديدي أو تحريضي.

\* محتوى ينتهك خصوصية الآخرين أو يتضمن بياناتهم دون إذن مناسب.

\* صور أو ملفات أو تسجيلات تخص أشخاصًا آخرين دون حق أو موافقة مناسبة.

\* محتوى احتيالي أو مضلل أو يهدف إلى الخداع أو الاستغلال.

\* محتوى يتضمن برمجيات ضارة أو روابط خطرة أو ملفات قد تضر بالأجهزة أو الأنظمة.

\* محتوى ينتهك حقوق الملكية الفكرية أو العلامات التجارية أو حقوق النشر.

\* محتوى غير لائق أو غير مناسب لطبيعة التطبيق الاجتماعية والتنظيمية.

\* أي إعلان أو عرض أو مزاد أو قائمة بيع أو خدمة مخالفة للأنظمة أو سياسات التطبيق أو متطلبات متاجر التطبيقات.



\---



\## 5. قواعد الدردشة والرسائل



تُستخدم الدردشة للتواصل بين أعضاء الديوانية. ويجب على المستخدم الالتزام بما يلي:



\* عدم الإساءة للأعضاء أو التشهير بهم أو مضايقتهم.

\* عدم إرسال رسائل مزعجة أو متكررة أو خارجة عن سياق الديوانية.

\* عدم مشاركة بيانات خاصة أو حساسة تخص الآخرين دون إذن.

\* عدم استخدام الدردشة للتهديد أو الابتزاز أو الاحتيال أو التضليل.

\* عدم إرسال ملفات أو روابط مشبوهة أو ضارة.



يحق للمؤسس أو المدير، وفق ما يتيحه التطبيق، اتخاذ إجراء تجاه المحتوى المخالف داخل الديوانية، كما يحتفظ التطبيق بحقه في اتخاذ الإجراء المناسب عند وجود مخالفة أو بلاغ أو طلب نظامي صحيح.



\---



\## 6. قواعد الصور والألبوم والملفات



عند استخدام الألبوم أو رفع الصور أو الملفات أو الصوتيات، يلتزم المستخدم بما يلي:



\* رفع محتوى يملك حق استخدامه أو مشاركته.

\* عدم نشر صور أشخاص أو مستندات أو بيانات خاصة دون إذن مناسب.

\* عدم رفع محتوى مخالف أو مسيء أو يضر بالآخرين.

\* عدم استخدام الملفات أو الصور للاحتيال أو التضليل أو انتهاك الخصوصية.

\* حذف أي محتوى أضافه إذا تبين أنه غير مناسب أو يخالف هذه السياسة.



يجوز للمؤسس أو المدير حذف بعض الصور أو الملفات المخالفة وفق الصلاحيات المتاحة، ولا يتحمل التطبيق مسؤولية المحتوى الذي يرفعه المستخدمون داخل الديوانيات، إلى الحد الذي تسمح به الأنظمة.



\---



\## 7. قواعد المقاضي والمصاريف المشتركة والتصويتات



\### 7.1 المقاضي



يجب استخدام المقاضي لتنظيم احتياجات الديوانية بشكل واضح وصحيح، وعدم إضافة عناصر مضللة أو غير مناسبة أو لا علاقة لها بالغرض التنظيمي للديوانية.



\### 7.2 المصاريف المشتركة



يجب على المستخدم تحري الدقة عند إضافة مصروف أو تسوية أو رصيد، ويكون مسؤولًا عن صحة البيانات التي يدخلها. المصاريف المشتركة داخل التطبيق سجل تنظيمي فقط، ولا تعد خدمة دفع أو تحويل مالي أو ضمان أو تحصيل من التطبيق.



لا يجوز استخدام المصاريف للإضرار بالأعضاء أو إضافة مبالغ غير صحيحة أو خلق التزامات مضللة. ولا يمكن إزالة عضو من الديوانية إذا كان له أو عليه رصيد قائم إلى حين تسوية الرصيد أو إغلاق الالتزامات المرتبطة به.



\### 7.3 التصويتات



يجب استخدام التصويتات لأغراض تنظيمية مناسبة داخل الديوانية، وعدم طرح تصويتات مخالفة أو مسيئة أو تنتهك خصوصية الآخرين أو تستخدم للتشهير أو الضغط أو التضليل.



\---



\## 8. قواعد السوق والعروض والمزادات



عند استخدام السوق أو القوائم أو العروض أو المزادات أو أي خاصية مشابهة، يلتزم المستخدم بما يلي:



\* عرض منتجات أو خدمات يملك حق عرضها أو التصرف فيها.

\* تقديم وصف صحيح وغير مضلل للمنتج أو الخدمة أو العرض.

\* عدم عرض منتجات أو خدمات ممنوعة أو مخالفة للأنظمة أو سياسات التطبيق.

\* عدم استخدام السوق للاحتيال أو التضليل أو رفع أسعار وهمية أو استغلال المستخدمين.

\* احترام حقوق الملكية الفكرية وحقوق الخصوصية وحقوق الأطراف الأخرى.

\* الالتزام بأي ضوابط إضافية تظهر داخل التطبيق.



أي تواصل أو اتفاق أو شراء أو بيع أو مزاد أو تسليم أو دفع يتم خارج التطبيق يكون على مسؤولية الأطراف المعنية وحدهم، ولا يتحمل التطبيق مسؤولية جودة المنتجات أو الخدمات أو الأسعار أو الضمانات أو الاسترجاع أو التسليم أو التزامات الأطراف خارج آلية معتمدة صراحة داخل التطبيق.



\---



\## 9. طلبات الانضمام والعضويات



يجب استخدام طلبات الانضمام ورموز الدعوة بطريقة مناسبة، ويُمنع:



\* مشاركة رموز الدعوة بشكل عشوائي أو بهدف الإزعاج أو الاستغلال.

\* إرسال طلبات انضمام متكررة أو مضللة.

\* استخدام أسماء أو بيانات غير صحيحة بقصد الخداع.

\* قبول أعضاء أو إدارتهم بطريقة تنتهك الأنظمة أو حقوق الآخرين.



يتحمل المؤسس والمدراء مسؤولية مراجعة طلبات الانضمام داخل ديوانيتهم واستخدام صلاحياتهم بحسن نية وبما يحقق مصلحة الديوانية ولا يخالف هذه السياسة.



\---



\## 10. مسؤولية المؤسس والمدراء



يمثل المؤسس والمدراء خط الحوكمة الأول داخل الديوانية. ويجب عليهم استخدام صلاحياتهم بمسؤولية، بما يشمل:



\* مراجعة طلبات الانضمام بعناية.

\* إدارة العضويات والصلاحيات بشكل عادل ومنظم.

\* عدم إساءة استخدام صلاحيات الترقية أو التنزيل أو الإزالة.

\* عدم حذف محتوى أو تقييد عضو بشكل تعسفي أو مخالف للأنظمة.

\* التعامل مع البلاغات أو المحتوى غير المناسب داخل الديوانية بقدر ما تسمح به وظائف التطبيق.



لا يعد المؤسس أو المدير ممثلًا قانونيًا أو تجاريًا عن التطبيق، ولا يملك صلاحية إلزام التطبيق بأي التزامات تجاه الأعضاء أو الأطراف الخارجية.



\---



\## 11. البلاغات والإجراءات



قد يوفر التطبيق وسائل للإبلاغ عن محتوى أو سلوك مخالف. كما يمكن التواصل معنا عبر البريد الرسمي عند الحاجة.



عند وجود مخالفة أو بلاغ جاد أو خطر أمني أو طلب نظامي صحيح، يحق للتطبيق اتخاذ إجراء مناسب، مثل:



\* حذف أو تقييد محتوى مخالف.

\* تقييد بعض الخصائص أو الصلاحيات.

\* تعليق أو تعطيل حساب أو ديوانية عند الحاجة.

\* إزالة عرض أو إعلان أو قائمة أو مزاد مخالف.

\* الاحتفاظ بسجلات محدودة لأغراض الحماية أو التحقيق أو الامتثال النظامي.

\* التعاون مع الجهات المختصة عند وجود أساس نظامي صحيح.



قد تختلف الإجراءات بحسب طبيعة المخالفة وتكرارها وخطورتها ومدى تأثيرها على المستخدمين أو الخدمة.



\---



\## 12. حدود مسؤولية التطبيق عن المحتوى



يقر المستخدم بأن المحتوى داخل الديوانيات ينشئه المستخدمون أنفسهم، وأن التطبيق لا يراجع كل محتوى قبل نشره. لذلك لا يتحمل التطبيق مسؤولية المحتوى أو التصرفات أو الاتفاقات أو البيانات التي ينشئها أو يشاركها المستخدمون، وذلك إلى الحد الذي تسمح به الأنظمة المعمول بها.



يقتصر دور التطبيق على توفير أدوات تقنية وتنظيمية، مع احتفاظه بحقه في اتخاذ الإجراءات المناسبة عند وجود مخالفة أو بلاغ أو طلب نظامي صحيح.



\---



\## 13. حماية الخصوصية والبيانات



يجب على المستخدم احترام خصوصية الآخرين وعدم مشاركة بياناتهم أو صورهم أو ملفاتهم أو معلوماتهم دون إذن مناسب. كما يجب عدم محاولة الوصول إلى بيانات أو ديوانيات أو محتوى لا يملك المستخدم صلاحية الوصول إليه.



يخضع التعامل مع البيانات داخل التطبيق إلى سياسة الخصوصية المعتمدة، ويقر المستخدم بأن بعض البيانات داخل الديوانية تكون مشتركة بين أعضائها بحسب طبيعة الخاصية.
