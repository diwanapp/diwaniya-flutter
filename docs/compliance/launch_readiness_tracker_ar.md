\## 1. الهدف من الملف



يهدف هذا الملف إلى متابعة الأعمال المتبقية قبل إطلاق تطبيق \*\*ديوانية\*\* بشكل تجاري رسمي، مع ترتيب الأولويات، تحديد الحالة، وتوثيق ملاحظات التنفيذ.



هذا الملف داخلي ولا يظهر للمستخدمين داخل التطبيق.

\## 4. اختبارات القبول قبل الإطلاق



\### 4.1 الحساب والخصوصية



| الاختبار | النتيجة المطلوبة | الحالة |

|---|---|---|

| حذف حساب بدون موانع | يتم الحذف ثم تسجيل الخروج | لم يختبر على production-like |

| حذف حساب عليه رصيد | يمنع الحذف وتظهر رسالة واضحة | لم يختبر على production-like |

| حذف حساب مدير وحيد | يمنع الحذف وتظهر رسالة واضحة | لم يختبر على production-like |

| فتح الخصوصية والشروط | تظهر الوثائق بشكل عربي مرتب | مكتمل محليًا |

| صفحة حذف الحساب الخارجية | تعمل وتعرض تعليمات واضحة | لم يبدأ |



\### 4.2 العضوية والصلاحيات



| الاختبار | النتيجة المطلوبة | الحالة |

|---|---|---|

| إزالة عضو عليه رصيد | يمنع الإزالة | مكتمل محليًا |

| إزالة عضو بدون رصيد | يسمح حسب الصلاحيات | يحتاج اختبار production-like |

| تغيير اسم عضو عليه رصيد ثم محاولة إزالته | يمنع الإزالة | يحتاج اختبار production-like |

| عضو عليه رصيد في ديوانية أخرى فقط | لا يمنع إجراءاته في ديوانية بلا رصيد | يحتاج اختبار production-like |

| مؤسس الديوانية | لا يمكن تنزيله أو إزالته من غيره | يحتاج اختبار regression |



\### 4.3 الأمن والإعدادات



| الاختبار | النتيجة المطلوبة | الحالة |

|---|---|---|

| API\_BASE\_URL في release | لا يحتوي localhost أو IP داخلي | لم يبدأ |

| HTTPS | مفعل في الإنتاج | لم يبدأ |

| OTP logs | لا تظهر OTP في سجلات الإنتاج | يحتاج تحقق |

| Tokens | لا تطبع في logs وتخزن آمنًا | مكتمل مبدئيًا |

| Rate limits | مفعلة على OTP والمسارات الحساسة | مكتمل مبدئيًا |
